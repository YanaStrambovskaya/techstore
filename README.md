# Test Task - TechStore page

This project is a test task to check layout, responsive design, and code skills.

## 🚀 Tech Stack

- Vite (build tool)
- HTML5 (semantic markup)
- CSS3 / Tailwind
- JavaScript (vanilla)

## 📱 Features

- Mobile-first responsive layout
- Adaptive design for desktop (based on mobile mockup)
- Pixel-perfect implementation
- Clean and structured code

## 🛠️ Setup & Run

1. Clone the repository:

```bash
git clone https://github.com/YanaStrambovskaya/techstore.git
```

2. Navigate to the project folder

```bash
cd techstore
```

3. Install dependencies
   Make sure you have Node.js installed (recommended version: 16+ or 18+).

```bash
npm install
```

This will install all required packages from package.json.

4. Start development server

```bash
npm run dev
```

The project will be available at: http://localhost:5173

5. Build the project for production

```bash
npm run build
```

This will generate an optimized production-ready build in the dist/ folder.

6. Preview production build locally (optional)

```bash
npm run preview
```
